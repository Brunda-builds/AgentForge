# 🚀 AgentForge: The Complete AI Hackathon Lifecycle Mentor
**Guiding freshers from zero knowledge to interview-ready projects.**

## 💡 The Vision
For a "fresher," the gap between learning a language and winning a hackathon feels like a mountain. **AgentForge** is the Sherpa that guides them every step of the way.

## 🛠️ The 5-Stage Mentorship Journey
AgentForge doesn't just answer questions; it follows a structured mentorship flow:

1.  **The Discovery Phase:** Discusses interests with the student to identify their "Spark."
2.  **Skill Mapping:** Suggests specific technical skills (Python, APIs, UI/UX) the student needs to develop based on those interests.
3.  **Ideation Engine:** If the student is stuck, AgentForge brainstorms unique hackathon project ideas tailored to their level.
4.  **Project Validator:** Once the student builds the project, the Agent reviews the logic, checks for correctness, and suggests improvements.
5.  **Interview Predictor:** Prepares the student for the "Judge's Table" by predicting tough technical and logic questions they might face during the presentation.

## 🏗️ Technical Stack
* **Language:** Python
* **Brain:** Google Gemini API (Generative AI) for adaptive mentorship.
* **Interface:** Streamlit for a seamless student experience.

## 🚦 Getting Started
1. Clone the repo: `git clone https://github.com/Brunda-builds/AgentForge.git`
2. Install: `pip install google-generativeai streamlit`
3. Run: `streamlit run agent.py`
